Install dependencies 
```
pip install -r requirements.txt
```
Run
```
python MazeSolver.py
```

To change a maze just write it in format.py and then type
```
python format.py
```
Now your maze is ready to be solved. \
Do it with
```
python MazeSolver.py
```
